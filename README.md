Vice City Games — Final Project (Completed)

An online video game store built as a single-page application (SPA) using HTML, CSS, and JavaScript with Maps and Objects.
Designed with a Miami Vice / neon aesthetic featuring themed tabs for each gaming platform.

Features

Platform Tabs

	•	PS5 — blue and pink gradient
	•	Xbox — neon green

	•	PC — cyberpunk grid

Shopping Cart
	•	Add and remove products

Search and Filters

Summary

Vice City Games is a completed front-end simulation of an online game store using Maps and Objects to manage catalog data, DOM manipulation for UI updates, and a cohesive Miami Vice-inspired design for a distinctive user experience. The project includes platform-based navigation, search and filter functionality, a working cart system, and an interactive chatbot.
